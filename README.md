Kernel Update
========================

This is for upgrading any android kernel source based on 3.4.5 JB source to 3.4.67 KK . 


How to use:

Step 1

    cd ~/routeyoursources

Step 2

    git clone https://github.com/Phoenix-Kernel/Kernel-Update patch

Step 3

    cd patch

Step 4

    chmod 777 patch && ./patch

Done. :)
